\tocsubsectionstar{Outline}

Being short sequences of characters, names are deceptively simple, yet they encode rich phonetic, morphological and cultural patterns that enable native speakers to immediately recognise their origin.\\

The aim of this project is to develop a model that is capable of generating novel toponyms (place names) and anthroponyms (personal names), steered at inference time by an initial set of example names. In other words, the model should be able to generate unseen sequences of characters that, to a native speaker, appear as pronounceable and plausible names from their culture, whether existing or not. Potential applications range from creating names for fictional worlds to devising names for branding and marketing purposes.\\

This task involves several challenges. The first challenge consists in creating a sufficiently robust, diverse, balanced, and well-labelled (culturally/linguistically) dataset of anthroponyms and toponyms.\\

The second challenge is to allow for sufficiently varied language forms, styles, and orthographies in the model outputs.\\


The third challenge would be to ensuring that output names are plausible names, but also phonetically/orthographically similar to the provided set of examples.

Finally, the fourth challenge, common across creative tasks, would be evaluating the model results.


Considerations such as prefix and suffix identification could be included, along with cultural alignment, the aesthetics of language generation, character set choice, and so on.\\
