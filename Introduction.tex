Names are deceptively simple - a handful of characters - yet they encode rich cultural, linguistic, and phonetic patterns that immediately signal origin and identity to a native speaker. The goal of this project is to produce a model that is able to generate new toponyms (place names) and anthroponyms (person names), guided at prediction time by an initial set of examples. Use cases could vary and include generation of names for fictional settings or generation of names for branding purposes.\\

A good model would be able to learn what makes a sequence of characters an anthroponym or a toponym from a given culture and to create new, albeit plausible, names from a given culture. This model should then be able to creatively come up with new names.\\

Considerations such as prefix and suffix identification could be included, along with cultural alignment, the aesthetics of language generation, character set choice, and so on.\\
The main challenges include: 
•	Creating a sufficiently robust, diverse, and well-labelled (culturally/linguistically) dataset of names (Wikidata is a proposed source).
•	Allowing for sufficiently varied language forms, styles, and orthographies in the model outputs.
•	Ensuring that output names are phonetically/orthographically similar to the provided set of examples (this is the crux of the problem).
•	Evaluating the model results (a common challenge in creative generation tasks), either as a user-based or statistical evaluation.
Depending on desired student focus, this project could be focussed entirely on ML/AI modelling and approaches, or potentially include a software engineering component of producing a web interface for the resulting model.\\
